UserStoriesTool

Real-time requirements traceability plugin for Eclipse

Download UserStoriesTool UserStoryLink.jar UserStoriesTool on GitHub

Setup

You need to install the .jar to enable the Annotation Functionality.

First, create a folder in the project and place the UserStoryLink.jar inside it. The Project Explorer should look like this:

Then right click on the project and select build-path.

Select Libraries->Classpath(depending on the eclipse version)->Add Jars and then select the UserStoryLink.jar

Creating an Annotation

To create a link between a UserStory and a java element (Method,Class or Constructor) just add the Annotation (@UserStoryLink) above the java element and press control + space (show content assist) and select a UserStory.

After saving the file the tool will create the link.

As you can see below, the annotation contains an array with the uids of the UserStories.

Example of a UserStory's uid : System.SubSystem.SubSystem.UserStory

Adding an anotation to a method will create the Annotation to the class automatically.
